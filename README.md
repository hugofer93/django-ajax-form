# djangoAjaxForm
Sample application to use forms with Ajax / Axios and Class-Based Views

* Django 2.2.4 LTS
* jQuery 3.3.1 Slim
* Bootstrap 4.3.1
* Datepicker for Bootstrap 1.6.4
* Axios 0.19

#### Note:
* Class-Based Views
* Render Form (GET Method)
* Submit (POST Method / AJAX)
* CSRF Token Handling (Axios)
* Error Handling and Validations
